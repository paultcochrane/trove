# trove

A program to store and lookup (encrypted) information

## Interactive mode

    $ trove
    > add
    > search
    > psearch
    > edit
    > del
    > clear
    > exit/quit

## Non-interactive mode

    $ trove --onecmd add     # add an entry
    $ trove --onecmd search  # search for an entry
    $ trove --onecmd delete  # delete an entry
